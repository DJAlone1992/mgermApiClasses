# /scheduledDoctors.php

Точка входа, для получения массива врачей, которые ведут прием по расписанию

## Параметры

Метод: GET
Параметры:
|Наименование|Обязательный|Тип|Формат|Описание|
|--|--|--|--|--|
|department||int| |Идентификатор отделения для выбора врачей только из данного отделения|
|specialty||int||Идентификатор специальности, для фильтрации|

## Ответ

В ответ получается массив врачей ведущих прием по расписанию [MgermApiClasses\Classes\Doctor](/src/Classes/Doctor.php)
